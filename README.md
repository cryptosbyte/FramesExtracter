# Frames extractor

A C++ program initially supposed to extract all frames from a video to a directory.

Install all necessary packages.
```
$ apt install g++
```

Run the MAKEFILE to compile and run the program with the `-h` option.
```sh
$ make compile
$ make run # for help
```

Initial run:
```sh
$ ./framer -h # or --help
```
